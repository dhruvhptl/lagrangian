Architecture

Modular encoder → shared latent embedding → discrete Lagrangian integrator → classifier head

Encoder (modular)

Four interchangeable encoders all accept (B, T=40, F) and return (B, encoder_dim). The best-performing variant is the causal Conv1D encoder, which preserves temporal order with no future leakage.


mlp — flatten + 2-layer MLP
conv1d — 2 causal conv layers, last-step readout
tcn — dilated causal TCN, dilations [1,2,4,8]
hybrid_conv — 3 layers F→64→128→64

Lagrangian Integrator

The encoder embedding is projected to an initial latent position z₀ and velocity ż₀. A discrete symplectic Euler integrator evolves this state for n steps under learned dynamics:


MassNet — diagonal positive mass M(q)
PotentialNet — scalar potential V(q) via autograd
DampingNet — position-dependent vector damping γ(q)
Update — velocity first, then position (symplectic)

Input (B, T=40, F)
      │
  [Encoder]  ←── mlp | conv1d | tcn | hybrid_conv
      │
      h  (B, encoder_dim)
   ┌──┴──┐
 z₀_head  ż₀_head
   │         │
  z₀        ż₀   (B, latent_dim)
      │
  [Symplectic Euler × n_steps]
    q    = coord_net(z)          optional coordinate transform
    V    = potential_net(q)      learned scalar potential
    dV   = ∂V/∂q                 autograd, create_graph=True during train
    M    = mass_net(q)           diagonal positive mass
    γ    = gamma_net(q)          vector damping
    z̈    = -(dV + γ·ż) / M
    ż    ← ż + dt·z̈             velocity update first (symplectic)
    z    ← z + dt·ż
      │
   z_T  (B, latent_dim)
      │
  LayerNorm → Linear(4)
      │
  logits (B, 4)

Quickstart

Tested on Python 3.11 · PyTorch · Hydra · yfinance

# Install

git clone https://github.com/dhruvhptl/lagrangian && cd lagrangian

pip install -r requirements.txt


# Run best Lagrangian model (subset first)

python -m src.training.train_lagrangian model=lagrangian_conv1d \

  +fold_start=20 +fold_end=40


# Full 71-fold benchmark

python -m src.training.train_lagrangian model=lagrangian_conv1d


# Run tests

python -m pytest tests/test_shapes.py -v
